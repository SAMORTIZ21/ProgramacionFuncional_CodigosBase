En esta Carpeta encontraras un juego basico.

Aplicamos lo siguiente:

1. Funciones Genericas
2. Un solo archivo Menus.fs que detecta el estado del programa (Program.fs) y con base a este imprime en pantalla el menu correspondiente, puede ser el menu de: GAME OVER, PAUSA o MENU PRINCIPAL
3. Se lleva acabo un sistema de pausado de programa sin borrar el estado actual de este, con el fin de que la pausa funcione correctamente sin reiniciar el programa.
4. Y se utiliza declaracion de variables Constantes al inicio del codigo con el fin de facilitar su edicion de funcionamiento.

IMPORTANTE:
Recordar que estos codigos son codigos de prueba, buscar tener una forma mayormente organizada
